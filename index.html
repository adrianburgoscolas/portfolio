Exchange Rate Calculator

Given the need of a tool to know the real time exchange rate from a variety of currencies, I built this Exchange Rate Calculator in three days, a web app developed with Nextjs, Typescript and conducted unit tests to ensure proper functioning. This app consume an API with a very limited number of request so I implemented a cache to reduce the number of request needed and extend the use of this tool improving the user experience of the client.

My Task List

The client asked for a Task List app using modern techniques, the text input should be formatted so adding colors to a hash tag(#ahasgtag), user mention(@user), an email(an@email.dev) and a link(www.alink.dev). Also needed persistent storage while adding tasks and be able to edit/delete the stored tasks. I implemented My Task List in 5 days, building the UI using React, Node/Express and implemented the storage with MongoDB using Atlas cloud services and conducted unit and integration tests to ensure proper functioning. I implemented a dynamic formatting behavior to the text input by adding colors as the client types and given patterns are detected adding an excellent user experience.

Pro Counter Web App

Pro Counter is an app to keep track of an inventory, a user can add, modify and delete any kind of items. A user can authenticate in the app to store his inventory persistently. I built this app using React for the UI, the backend was built with Go(lang) without using any framework/library. I also implemented the authentication from scratch, to keep the session I used JWT(json web token) so the microservice can scale well to be used from a non web browser UI.

Gateways API

Given the necessity to have a register to keep track of a network devices I built Gateway API using React, Node/Express and MongoDB Atlas cloud service for the persistent storage this is a REST micro-service with a simple UI so the client can use the service UI or any other UI, the client can keep track of its network gateways by adding, modifying or deleting them from a persistent storage, the client can also add several devices connected to every gateway.

Dsync

One way to keep you sensitive data away from ransomware or any other kind of data loses is to backed up to a cloud storage. I built DSync, a CLI app to keep your files and folders synced to you own Google Drive storage. I built this CLI app with Go/Golang and integrated the app with Google Drive API so the user can use their own Google Storage to keep his data safe manual or automatic, the user can schedule the syncing task every amount of minutes adding a security layer to its data.

Control Demanda

The Cuba's Electric Power Company penalty any client having an inefficient electric power consumption by applying an extra fare ricing up its expenses. Control Demanda is a web app I'm currently working on using React, Node/Express and SQLite over a Raspberry Pi platform. The data is collected every 10 minutes from an electric network analyzer and stored in the platform, then using a system of graphics the client is aware of its electrics parameters on real time . This app can help any client to monitor and control its electric power demand and parameters so any electric power consumption deficiency can be fixed saving money to the client and been more sustainable with the environment.
